📋 Setup instructions

This page uses the Google Calendar API. To connect your account:

1. Open Google Cloud Console
2. Create a project → Enable Google Calendar API
3. Go to Credentials → Create OAuth 2.0 Client ID (Web application)
4. Add http://localhost or your file path to Authorized JS Origins
5. Copy your Client ID and paste it below, then save the file.
